branching

New branch

git checkout -b <new-branch>

push branch on remote

git push -u origin <branch>

switch branch

git checkout <branch>

history

check old commits and get their hash

git log

go back in time (definitive)

git reset --hard <commit-hash>

go back in time

git revert <commit-hash>

git stash
